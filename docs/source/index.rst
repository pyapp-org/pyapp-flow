######################################
Welcome to pyApp Flow's documentation!
######################################

pyApp Flow is a simple but powerful application level workflow library.

Allows complex processes to be broken into smaller specific steps, greatly
simplifying testing and re-use.

Installation
============

Python >=3.8 is required

.. code-block:: shell

  pip install pyApp-Flow

Next steps run through the :doc:`getting-started` guide.
